How to play ?

Fill cells with numbers from 1 to 9

The board has 9 regions, each has 9 cells

Rule 1 - each Region consist of unique numbers (1-9)

Rule 2 - each Row of the board consist of unique numbers (1-9)

Rule 3 - each Column of the board consist of unique numbers (1-9)

How to use ?

Type in each square a number

Remove number to empty square

Start a new easy/medium/hard board

Stucked? Solve the board

Editor mode

You can create your own board and save it

Start with a random board

Remove/add numbers while constant checking for unique solutions

Board is saved in a string format where "." means empty

Or you can load a board (using the same string format)

Solve, remove, add, load, save as you wish

How it works ?

Empty board is generated

First row is filled with random numbers

Board is populated with numbers until it's a valid Sudoku game

Random cells are removed from board

Difficulty level is described by the number of filled squares

Each board is probably unique by the laws of probability

The game generates new boards in the background using WebWorkes

MORE boring facts

The name "Sudoku" stems from two Japanese words: "su", which means "number" and doku, which means "single" Translated, it mostly means "single numbers only"

Actually, Sudoku isn't a Japanese game it all. It's American invented. Howard Garns created it as Number Place in 1979 but died in 1989 before Japanese publisher Nikoli got a hold of it

British Airways absolutely forbids its flight attendants from solving Sudoku puzzles during either takeoff or landing

There are 6,670,903,752,021,072,936,960 possible combinations for completing a 9-by-9 Sudoku grid, but only 5,472,730,538 of them really count for different solutions. Needlessly to say, you need a handful of lifetimes to solve all of them

When Sudoku became a world hit in 2005, it is estimated that it is the biggest phenomenon since the Rubik's Cube in the 1980
